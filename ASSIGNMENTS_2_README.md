## Task 2
Convert the following to hex format:
e.g: rgb(123,56,245) = #7b38f5
123 = 16 * 7 + 1 * 11 = 7b
56  = 16 * 3 + 1 * 8  = 38
245 = 16 * 15 + 1 * 5 = f5

* rgb(12,228,113) = #0ce471
12 = 16 * 075 + 1 * 0 = 0c
228 = 16 * 14 + 1 * 4 = e4
113 = 16 * 7 + 1 * 1 = 71

* rgb(12,226,25) = #0ce219
12 = 16 * 0.75 + 1 * 0 = 0c
226 = 16 * 14 + 1 * 2 = e2
25 = 16 * 1 + 1 * 5 = 19

* rgb(112,124,21) = #707c15
112 = 16 * 7 + 1 * 0 = 70
124 = 16 * 7 + 1 * 12 = 7c
21 = 16 * 1 + 1 * 5 = 15

* rgb(213,132,13) = #d5840d
213 = 16 * 13 + 1 * 0 = d5
132 = 16 * 8 + 1 * 0 = 84
13 = 16 * 1 + 1 * 0 = 0d

* rgba(112,223,14,.4) = #70df0e
112 = 16 * 7 + 1 * 0 = 70
223 = 16 * 13 + 1 * 0 = df
14 = 16 * 1 + 1 * 0 = f0
.4 = 16 * 0.66 + 1 * 0 = e
